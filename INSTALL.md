# Analytical Dashboard — Орнату және іске қосу нұсқаулығы (Windows)

---

## 1. Қажеттіліктер

- **Node.js** (https://nodejs.org/)
- **Docker Desktop** (https://www.docker.com/products/docker-desktop/)

---

## 2. Орнату қадамдары

1. Node.js және Docker Desktop орнатылғанына көз жеткізіңіз.
2. Папканы кез келген жерге көшіріңіз (мысалы, `C:\analytical-dashboard`).
3. `.env.example` файлын `.env` деп өзгертіңіз (немесе `setup.bat` өзі көшіреді).
4. `setup.bat` файлын әкімші ретінде іске қосыңыз (оң жақ батырма → Run as administrator).
5. Барлық процесс автоматты түрде жүреді. Соңында браузерде http://localhost:3000 ашыңыз.

---

## setup.bat не істейді?

- .env файлын автоматты түрде жасайды (егер жоқ болса)
- npm тәуелділіктерін орнатады
- Prisma client-ті генерациялайды
- Миграцияны базаға жібереді
- Docker арқылы PostgreSQL-ды іске қосады
- Серверді автоматты түрде іске қосады

---

## Қолмен іске қосу (егер setup.bat жұмыс істемесе)

1. `.env.example` файлын `.env` деп көшіріңіз.
2. `npm install` орындаңыз.
3. `npm run db:generate` орындаңыз.
4. `npm run db:push` орындаңыз.
5. `docker compose up -d` арқылы дерекқорды іске қосыңыз.
6. `npm run dev` арқылы серверді іске қосыңыз.

---

## Ескертулер

- Егер порт 3000 бос болмаса, `.env` ішіндегі `NEXTAUTH_URL`-ды өзгертіңіз.
- Docker іске қосылуы үшін Windows-та WSL2 немесе Hyper-V қосулы болуы керек.
- Егер сұрақтар туындаса, README.md немесе AGENTS.md қараңыз.
- `.env` файлы міндетті, үлгісі `.env.example`-де бар.
- Егер автоматты скрипттермен қиындық болса, қолмен орнату нұсқауын қолданыңыз.
